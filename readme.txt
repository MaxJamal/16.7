 Задание 16.7.1

Откройте сайт "Дом питомца" и на основе имеющихся в нем данных создайте конструктор класса Dog
со следующими параметрами: имя, пол, возраст.

В отдельный файл импортируйте и создайте объект Dog, который выводит имеющихся на сайте котов,
с одинаковыми параметрами, но с разными значениями.

Исходники с кодом выкладывайте в репозиторий Github, призывайте для проверки ментора в Slack.
